 src/index.jsx ให้ใช้ฟังก์ชัน fetch() ในการดึงข้อมูลจากไฟล์ courses.json มาแสดงผล
 สิ่งที่เปลี่ยนแปลงในโค้ด:
   1. ใช้ useState และ useEffect: สร้าง State สำหรับเก็บข้อมูลคอร์ส (courses) และสถานะการโหลดข้อมูล (loading)
   2. ฟังก์ชัน fetch(): เรียกใช้ fetch("/src/data/courses.json") เพื่อดึงข้อมูลแบบ Asynchronous (เหมือนการดึงข้อมูลจาก Server จริง)
   3. ระบบ Loading: เพิ่มการแสดงผล Loading Spinner (วงล้อหมุนสีแดง) ในขณะที่รอกำลังโหลดข้อมูล เพื่อให้ User Experience ดูดีขึ้น
   4. Error Handling: มีการเช็ค response.ok และ catch error เพื่อความปลอดภัยของระบบ

   3. สร้างส่วน Filter Panel:

       * ราคา: สไลด์เพื่อปรับช่วงราคาที่ต้องการ (Price Range)
       * ปุ่มล้างค่า: สำหรับรีเซ็ตตัวกรองทั้งหมดกลับเป็นค่าเริ่มต้น
   4. ระบบการกรองข้อมูล (Real-time Filtering):
       * หน้าเว็บจะอัปเดตรายการคอร์สทันทีที่คุณพิมพ์ค้นหา เลือกหมวดหมู่ หรือปรับราคา
       * มีการแสดงจำนวนคอร์สที่พบ (เช่น "คอร์สเรียนที่ตรงตามเงื่อนไข (12)")
   5. ปรับปรุง UI ของ Course Card:
       * เพิ่ม Badge แสดงหมวดหมู่ (มุมซ้ายบน) และป้ายราคา (มุมขวาล่าง) บนรูปภาพเพื่อให้ดูข้อมูลได้ง่ายขึ้น
